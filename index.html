Informasi Saya

Nama saya mustofa, Saya lahir di jakarta tahun 1997, memasuki bangku sekolah dasar di tahun 2004, sekolah daasar saya sama dengan anak yang lain, hoby saya saat sekolah dasar adalah berlari, Saat lomba 17 agustus saya berusaha manjadi nomor satu dilomba lari, Saya berhasil juara satu saat saya kelas lima sekolah dasar, saya tidak terlalu paham tentang arti belajar saat bangku sekolah dasar, setalah lulus saya menjoba mendaftarkan diri ke-sekolah negri, saya tidak lolos di glombang pertama tapi saya lolos gelombang kedua dengan peruntungan saya, Saya dibangku sekolah menengah pertama ditahun 2010, saya mulai mengeti apa itu belajar, pelajaran yang mebuat saya tertarik adalah matematika walupun itu pelajaran sulit untuk saya.
saya menganal nama konsep - konsep logika dasar, tapi itu masih sulit untuk saya, setelah lulus dari sekolah menengah pertama, saya masuk ke-sekolah swasta di tahun 2013, di bangku sekolah menengah kejuruan saya tertarik dengan namanaya sistem belajar melalui praktik, setelah lulus saya mencari pekrjaan dan sembaring kuliah, lambatlau mata saya mulai mengalami rabun jauh, saya tertarik dengan sistem belajar praktik, matematika, dan mata rabun jauh tidak menggangu belajar saya maka dari itu saya memilih Jurusan komputer, dari semua pelajaran saya tertarik memabuat aplikasi dan tugas akhir saya membuat aplikasi android menggunakan java di android studio.
